# DelStrat Weekly Report Prototype

An HTML prototype for the Deliverability Strategy weekly report view, built by the Amplify CSM team.

## How to use

Open `weekly-report.html` directly in a browser — no build step or server needed.

## Contributing

1. Clone the repo
2. Create a branch off `main` (`git checkout -b your-branch-name`)
3. Make your changes to `weekly-report.html`
4. Open a pull request into `main`

## Project context

This prototype supports the DelStrat design sprint. When the project is complete, this repo will be archived.
